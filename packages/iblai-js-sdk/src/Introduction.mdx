# IBL AI SDK

Welcome to the IBL AI SDK component documentation!

## What is IBL AI SDK?

The IBL AI SDK is a TypeScript SDK for building AI-powered educational applications with the IBL AI Platform. It provides:

- **Data Layer**: RTK Query API slices and hooks for data fetching
- **Web Utils**: React providers, hooks, and utilities
- **Web Containers**: React UI components for web applications
- **Native Components**: React Native components for mobile apps

## Installation

```bash
npm install @iblai/iblai-js-sdk
# or
yarn add @iblai/iblai-js-sdk
# or
pnpm add @iblai/iblai-js-sdk
```

## Quick Start

```tsx
import { useGetMentorsQuery, AuthProvider } from '@iblai/iblai-js-sdk';

function App() {
  return (
    <AuthProvider>
      <MentorsList />
    </AuthProvider>
  );
}

function MentorsList() {
  const { data, isLoading } = useGetMentorsQuery({
    org: 'my-org',
    limit: 10,
  });

  if (isLoading) return <div>Loading...</div>;

  return (
    <div>
      {data?.results.map((mentor) => (
        <div key={mentor.id}>{mentor.name}</div>
      ))}
    </div>
  );
}
```
